    Membuat Text Typing Animation Menggunakan JavaScript

    March 08, 2026

    Pernah melihat tulisan di website yang muncul seperti sedang diketik oleh komputer? Hurufnya muncul satu per satu, kadang disertai cursor berkedip di ujung teks. Efek seperti ini sering dipakai di landing page, portfolio developer, atau website teknologi karena terlihat lebih hidup dan interaktif.

    Menariknya, efek seperti ini ternyata tidak perlu library berat atau framework khusus. Cukup menggunakan HTML, CSS, dan sedikit JavaScript, kita sudah bisa membuat animasi teks yang terlihat profesional.

    Pembukaan


    Pada artikel kali ini di RJ Code. Serta Seri Best JavaScript part 3, kita akan mencoba membuat Text Typing Animation sederhana menggunakan JavaScript. Efek ini akan menampilkan teks yang muncul huruf demi huruf, seperti sedang diketik secara langsung di layar.

    Contoh yang akan kita buat nantinya kurang lebih seperti ini :

    Welcome to RJ Code

    Learn Coding With Simple Project

    Best JavaScript Animation

    Setiap kalimat akan muncul secara perlahan dan otomatis turun ke baris berikutnya. Efek ini sangat cocok digunakan pada hero section website, halaman portfolio, atau bahkan landing page produk.

    Mengapa Fitur Ini Penting Untuk Dibuat


    Walaupun terlihat sederhana, fitur text typing animation sebenarnya punya peran yang cukup penting dalam tampilan website. Tanpa animasi, teks biasanya hanya tampil statis dan terlihat biasa saja. Namun ketika ditambahkan efek mengetik, halaman website terasa lebih hidup dan menarik perhatian pengunjung.

    Selain itu, efek ini sering digunakan untuk menonjolkan pesan utama di bagian atas website. Misalnya memperkenalkan siapa kita, apa yang kita buat, atau layanan apa yang ditawarkan.

    Contoh video : Video source by : Pinterest

    Banyak website portfolio developer bahkan menggunakan efek ini untuk menampilkan kalimat seperti :

    I am Web Developer

    I am UI Designer

    I build Modern Website

    Karena tampilannya simpel tapi tetap terlihat profesional.

    Manfaat Pembuatan Fitur Tersebut


    Selain membuat tampilan website lebih menarik, ada beberapa manfaat lain dari fitur ini. Pertama tentu saja dari sisi user experience, pengunjung tidak langsung disuguhi teks panjang tetapi melihat animasi yang lebih interaktif.

    Efek ini juga bisa menjadi fokus perhatian di bagian hero website, sehingga pengunjung langsung membaca pesan utama yang ingin kita tampilkan.

    Dari sisi pembelajaran coding, membuat fitur ini juga melatih kita memahami beberapa konsep penting di JavaScript seperti array, pengulangan teks, manipulasi DOM, dan penggunaan setTimeout untuk membuat animasi sederhana.

    Cara Pembuatan Secara Simple dan Praktis


    Cara membuatnya sebenarnya cukup sederhana. Pertama kita hanya perlu menyiapkan elemen HTML yang akan menampilkan teks animasi. Biasanya cukup menggunakan satu elemen seperti heading atau div yang memiliki id tertentu, misalnya id typing.

    Setelah itu kita menambahkan JavaScript yang berfungsi untuk mengambil teks dari beberapa kalimat yang sudah kita simpan di dalam sebuah array. Script ini kemudian akan mengambil huruf satu per satu dari kalimat tersebut dan menampilkannya secara bertahap ke dalam elemen HTML.

    Setiap huruf akan muncul dengan jeda waktu tertentu menggunakan fungsi setTimeout, sehingga terlihat seperti sedang diketik secara manual. Setelah satu kalimat selesai ditampilkan, script akan menambahkan baris baru lalu melanjutkan mengetik kalimat berikutnya.

    Dengan cara ini kita bisa membuat beberapa kalimat tampil secara bertahap tanpa harus menggunakan plugin tambahan. Semua animasi bisa berjalan hanya dengan JavaScript sederhana.

    Kesimpulan


    Text Typing Animation adalah salah satu efek JavaScript yang sederhana namun cukup powerful untuk meningkatkan tampilan website. Dengan sedikit kode JavaScript, kita bisa membuat teks yang biasanya statis menjadi lebih hidup dan menarik perhatian pengunjung.

    Selain mempercantik tampilan website, fitur ini juga membantu kita memahami bagaimana cara JavaScript berinteraksi dengan elemen HTML dan mengatur waktu animasi menggunakan fungsi sederhana.

    Penutup


    Itulah cara membuat Text Typing Animation menggunakan JavaScript yang bisa langsung kamu terapkan di website. Efek ini sangat cocok digunakan pada landing page, portfolio developer, ataupun halaman utama website agar terlihat lebih modern.

    Jika kamu tertarik mempelajari lebih banyak efek menarik lainnya, kamu bisa terus mengikuti artikel terbaru di RJ Code, karena akan ada banyak tutorial JavaScript sederhana yang bisa langsung kamu praktikkan.
